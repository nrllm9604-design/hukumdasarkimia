# hukumdasarkimia
ChemBound adalah platform pembelajaran kimia digital yang dirancang khusus untuk membantu siswa kelas X menguasai materi Hukum Dasar Kimia secara mandiri, seru, dan mendalam. Platform ini mengubah teori kimia yang abstrak menjadi visualisasi interaktif yang mudah dipahami tanpa rumus yang menjemukan.
